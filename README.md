# 💻 Sistema de Cadastro de Alunos (CRUD)

Este projeto foi desenvolvido como atividade prática para a disciplina de **3DAW** da faculdade. Trata-se de um mini-sistema CRUD focado em uma interface amigável e fluida.


## ⚙️ Funcionalidades

* **Create:** Adição de novos alunos via formulário.
* **Read:** Listagem dinâmica dos alunos cadastrados.
* **Delete:** Exclusão individual de registros utilizando IDs únicos gerados pelo PHP.

## 🛠️ Tecnologias Utilizadas

* **PHP 8+** (Lógica principal e controle de Sessões)
* **HTML5** (Estruturação)
* **CSS3** (Estilização avançada)
